# Karlos-Peluqueria-Updload-Form
Formulario para subir las fotos de los perros a la web de Karlos (peluquería canina)


**Nombre**

- [ ]  ¿Es un campo rellenado? (Obligatorio)
- [ ]  ¿Empieza por mayúscula?
- [ ]  ¿Contiene solo letras? (Sin números ni símbolos)

**Apellidos**

- [ ]  ¿Es un campo rellenado? (Obligatorio)
- [ ]  ¿Empiezan por mayúscula?
- [ ]  ¿Contiene solo letras? (Sin números ni símbolos)

**Dirección**

- [ ]  ¿Es un campo rellenado? (Obligatorio)
- [ ]  ¿Sigue el orden: `Tipo vía/ Nombre vía, Número, Resto, CP, Población, País`?
- [ ]  ¿Están los elementos correctamente separados por comas?

**Email**

- [ ]  ¿Es un campo rellenado? (Obligatorio)
- [ ]  ¿Tiene estructura `dirección@nombreDominio.RestoDominio`?
- [ ]  ¿Usa solo caracteres permitidos? (letras, números, puntos y guiones bajos)

**Confirmar Email**

- [ ]  ¿Es un campo rellenado? (Obligatorio)
- [ ]  ¿El formato es correcto?
- [ ]  ¿Coincide exactamente con el primer email introducido?

**DNI**

- [ ]  ¿Es un campo rellenado? (Obligatorio)
- [ ]  ¿Supera el algoritmo de validación oficial?

**Intereses**

- [ ]  ¿Se han separado los elementos por comas?
- [ ]  ¿Hay al menos un interés válido entre los introducidos?
- [ ]  ¿Se han filtrado y eliminado los intereses incorrectos (si los hay)?

**Password**

- [ ]  ¿Es un campo rellenado? (Obligatorio)
- [ ]  ¿Tiene entre 8 y 20 caracteres?
- [ ]  ¿Contiene al menos 1 mayúscula?
- [ ]  ¿Contiene al menos 1 minúscula?
- [ ]  ¿Contiene al menos 2 números?
- [ ]  ¿Contiene al menos 1 símbolo?
